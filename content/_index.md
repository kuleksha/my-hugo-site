Проект посвящен разработке ПО для 3D-принтеров с системой многоматериальной печати, которое автоматизирует работу нескольких экструдеров. Программа поддерживает форматы STL, OBJ, 3MF и позволяет гибко настраивать параметры для каждого типа материала. Благодаря интеллектуальной синхронизации обеспечивается бесшовное переключение между материалами и высокое качество готовых изделий. Система включает модули генерации G-code, визуализации печати и встроенную диагностику устройства. Цель проекта — сделать сложную многокомпонентную печать простой и доступной для любого пользователя.
